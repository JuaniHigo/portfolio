Diseño de Base de Datos para Guardería Náutica

Descripción General

Este proyecto académico se centró en el diseño completo de una base de datos relacional para gestionar las operaciones de una guardería náutica, desde el registro de clientes y embarcaciones hasta la facturación y gestión de servicios.

Objetivo

El objetivo era crear un modelo de datos normalizado y eficiente que asegurara la integridad de la información y permitiera realizar consultas complejas para la toma de decisiones operativas y gerenciales.

Paso a Paso

  1. 1. Relevamiento de los requerimientos del sistema y definición de las entidades principales.
  2. 2. Creación del Diagrama Entidad-Relación (DER) para visualizar las tablas y sus relaciones.
  3. 3. Aplicación de las formas normales (hasta 3FN) para eliminar redundancias y mejorar la consistencia de los datos.
  4. 4. Escritura de las sentencias DDL (Data Definition Language) en SQL para la creación de la estructura de la base de datos en MySQL.

Tecnologías Utilizadas

MySQL
Modelado de Datos
