Diseño de Base de Datos para Guardería Náutica

Este proyecto abarca el ciclo de vida completo del diseño de una base de datos relacional para un sistema de gestión de una guardería náutica, incluyendo una escuela de actividades acuáticas.

A partir de los requerimientos de negocio, se desarrolló un modelo conceptual, un modelo lógico y finalmente la implementación física en SQL.

Proceso de Diseño

El desarrollo siguió un proceso estructurado, comenzando por el análisis de los requerimientos para identificar entidades y relaciones clave. Posteriormente, se construyó un Diagrama Entidad-Relación (DER) para modelar conceptualmente el sistema. Este modelo fue luego transformado a un esquema relacional, aplicando principios de normalización para asegurar la integridad de los datos. La etapa final consistió en la escritura de un script SQL con las sentencias DDL y DML para crear e inicializar la base de datos.
