Generadores y Tests de Aleatoriedad

Descripción General

Este proyecto se centra en la implementación y evaluación de algoritmos generadores de números pseudoaleatorios, como el Método Congruencial Lineal y el Método Congruencial Mixto.

Objetivo

El objetivo era comprender cómo funcionan estos generadores y validar la "calidad" de su aleatoriedad mediante la aplicación de pruebas estadísticas rigurosas, como el Test de Chi-Cuadrado y el Test de rachas.

Paso a Paso

  1. 1. Implementación de los algoritmos de generación de números en Python.
  2. 2. Desarrollo de un conjunto de pruebas estadísticas para evaluar la uniformidad e independencia de las secuencias generadas.
  3. 3. Generación de grandes volúmenes de datos para realizar las pruebas de manera efectiva.
  4. 4. Análisis de los resultados para determinar si los generadores pasan las pruebas de aleatoriedad y documentación de las conclusiones.
