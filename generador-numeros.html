Simulación de Distribuciones de Probabilidad

Este proyecto se enfoca en la implementación de dos métodos fundamentales para la generación de variables aleatorias que siguen una distribución de probabilidad específica: el Método de la Transformada Inversa y el Método de Rechazo.

El objetivo es demostrar la capacidad de generar datos que se comporten como si provinieran de fenómenos reales modelados por distribuciones como la Normal, Exponencial, Gamma, Uniforme, entre otras. Posteriormente, se aplican pruebas de bondad de ajuste como Anderson-Darling y Kolmogorov-Smirnov para validar estadísticamente que los datos generados se ajustan correctamente a los modelos teóricos.

Proceso y Metodología

Para cada distribución, el script genera un conjunto de datos y produce una visualización comparativa que incluye un histograma de los datos generados y la curva de la Función de Densidad de Probabilidad (PDF) teórica. Adicionalmente, se incluyen gráficos Q-Q para una comparación visual más rigurosa de los cuantiles.
