Simulación de Distribuciones

Descripción General

Este proyecto consiste en una herramienta desarrollada en Python para generar números pseudoaleatorios que se ajustan a nueve distribuciones de probabilidad diferentes, tanto continuas como discretas.

Objetivo

El objetivo principal era aplicar y validar métodos de generación de variables aleatorias, como el Método de la Transformada Inversa y el Método de Rechazo, y luego verificar la correcta implementación mediante pruebas de bondad de ajuste.

Paso a Paso

  1. 1. Investigación de los métodos de generación para cada distribución.
  2. 2. Implementación de los algoritmos en Python utilizando librerías como NumPy.
  3. 3. Desarrollo de pruebas de bondad de ajuste (Chi-Cuadrado y Kolmogorov-Smirnov).
  4. 4. Creación de visualizaciones con Matplotlib para comparar los histogramas generados con las funciones de densidad teóricas.
