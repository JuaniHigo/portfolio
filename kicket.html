Kicket - Venta de Entradas Online

Descripción General

Kicket es una aplicación web full-stack diseñada para la venta de entradas a eventos. Permite a los usuarios explorar eventos, seleccionar asientos y realizar compras de manera segura, mientras que los administradores pueden gestionar eventos, precios y localidades.

Objetivo

El objetivo era construir una aplicación web completa y funcional, aplicando tecnologías modernas del stack MERN (MongoDB, Express, React, Node.js) y demostrando habilidades en el desarrollo tanto del frontend como del backend.

Paso a Paso

  1. 1. Diseño de la interfaz de usuario y la experiencia del usuario (UI/UX).
  2. 2. Desarrollo del frontend utilizando React para crear una interfaz interactiva y responsive.
  3. 3. Construcción de la API REST en el backend con Node.js y Express para manejar la lógica de negocio y la comunicación con la base de datos.
  4. 4. Integración del frontend y el backend, y despliegue de la aplicación.
