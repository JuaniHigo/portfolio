Simulación de Sistemas con AnyLogic y Python

Descripción General

Este proyecto aborda la simulación de un sistema de gestión de inventarios para una empresa, utilizando el software especializado AnyLogic y complementándolo con análisis de datos en Python.

Objetivo

El objetivo era modelar el comportamiento de la demanda y la logística de un producto para determinar la política de inventario óptima que minimice los costos totales, considerando costos de almacenamiento, de orden y de quiebre de stock.

Paso a Paso

  1. 1. Análisis de datos históricos de demanda en Python para identificar la distribución de probabilidad más adecuada.
  2. 2. Construcción del modelo de simulación en AnyLogic, representando el flujo de productos, políticas de reorden y costos asociados.
  3. 3. Ejecución de múltiples escenarios de simulación variando los parámetros de la política de inventario.
  4. 4. Exportación de los resultados y análisis final en Python para identificar la configuración óptima y presentar las conclusiones.

Tecnologías Utilizadas

Python
AnyLogic
Simulación
